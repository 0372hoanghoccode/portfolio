## Overview

Movie-Chill is a movie discovery platform built with **Next.js 16** and **Firebase**. Deployed on **Cloudflare** via OpenNext for edge performance. Reached 400+ daily active users and 600+ registered accounts.

## Key Features

- Browse and search movies/series with real-time search
- User authentication (Firebase Auth)
- Personal watchlist & favorites
- Smooth animations with Framer Motion
- Form validation with React Hook Form + Zod

## Technical Highlights

- **Next.js 16** App Router with Turbopack dev server
- **Zustand** for global state, **TanStack Query** for server state
- **Radix UI** + shadcn/ui component system
- **Vitest** + **Playwright** for unit and E2E testing
- Deployed to **Cloudflare** via `@opennextjs/cloudflare`
- Firebase Auth + Firestore for user data (100% free stack)
