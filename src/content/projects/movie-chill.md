## Overview

Movie-Chill is a full-featured movie discovery and streaming platform built with **Next.js** and powered by **Firebase** on the backend. The project gained real traction with 400+ daily active users and 600+ registered accounts.

## Key Features

- Browse and search movies/series with real-time search
- User authentication (Firebase Auth)
- Personal watchlist & favorites
- Responsive UI praised by users for excellent UX
- Firebase Firestore for user data and preferences

## Technical Highlights

- **Next.js** App Router for SSR/ISR performance
- **Firebase** for auth, database (Firestore), and hosting
- Optimized image loading and lazy hydration
- Mobile-first responsive design
